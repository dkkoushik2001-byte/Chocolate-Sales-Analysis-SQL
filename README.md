#  Chocolate Sales Analysis | SQL Project

## 📌 Project Overview
This project analyzes chocolate sales data using MySQL.  
The main objective is to identify sales performance, customer behavior, product trends, and profit growth using SQL queries.

This project is created for practicing:
- SQL Queries
- Data Cleaning
- Data Analysis
- Business Insights
- Reporting

# 🎯 Project Objectives

- Analyze total sales and profit
- Identify top-selling products
- Find brand-wise revenue
- Track monthly growth percentage
- Analyze customer segmentation
- Measure loyalty member performance
- Generate business insights using SQL
